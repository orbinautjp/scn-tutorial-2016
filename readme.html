このファイルについて

このファイルは、Orbiter日本語チュートリアルのためのサンプルシナリオです。
Orbiter 2016で使用できます。

チュートリアルの途中からゲームを開始することができます。
特定のPartだけを練習する、自分の軌道との答え合わせをする、などの目的に使用できます。

インストール

Scenariosフォルダをコピーして、Orbiterのインストールフォルダに上書き。

使用方法

Orbiterを起動する。
_Tutorials_JP_2016フォルダをダブルクリック。
使用したいシナリオをダブルクリックして、ゲームを開始する。

  • Orbiterの基本操作

注意点

Orbiterでは、MFDなどの状態を完全にセーブすることはできません。
TransXの場合、ViewやStageは保存されないので、ゲーム開始後に切り替える必要があります。

ゲーム開始時のロード時間短縮のため、不要な宇宙船などを削除しています。
そのため、Map MFDなどの表示がチュートリアルと異なることがあります。

不具合などを発見したときは、以下のページで報告してください。
